# 前言:
這篇主要是記錄我在台中人工智慧學校助教培訓第二周作業的學習，題目是Mnist手寫數字辨識。<br>
## 認識 MNIST 手寫數字辨識
對於剛開始接觸 TensorFlow 、 Keras 等深度學習框架的人而言，「MNIST 手寫數字辨識」是非常適合入門的練習項目，其角色就好比剛學習程式語言時所印出的 " Hello world ! " 一樣，讓新手初步認識與練習 TensorFlow。<br>

而 MNIST 手寫數字辨識，顧名思義，便是讓機器辨識出手寫數字影像是 0 到 9 之間的哪一個數字，究竟這種對於人類看似輕而易舉的事情，交由機器來做的表現又會如何呢？接下來，我們將會簡單介紹如何利用 TensorFlow 完成這項練習，藉此熟悉 TensorFlow 的基本運用。
